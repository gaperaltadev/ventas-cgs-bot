Antes de empezar

El CGS Bot es un asistente de WhatsApp para el equipo de ventas. Te permite consultar el catálogo completo de productos YPF, registrar ventas y ver reportes — sin instalar nada, desde tu celular.

Cómo agregarlo a tus contactos

  1. El administrador te da el número del bot.
  2. Guardalo en tu celular con un nombre que recuerdes (por ejemplo: CGS Bot).
  3. Abrí WhatsApp y mandale /ayuda.
Importante: el bot solo responde a los números autorizados. Si te aparece el mensaje "Este bot es de uso interno de CGS Paraguay", pedile al admin que te dé acceso.
2

La regla básica: el prefijo /

Para que el bot te responda, los comandos deben empezar con una barra diagonal (/). Esto evita que el bot se confunda con conversaciones normales.

Tú: /catalogo
Bot: 📋 Catálogo CGS Paraguay
    ...lista de productos...
Tú: hola que tal
Bot: (no responde — falta el /)
Excepción: cuando el bot está esperando una respuesta tuya (te pidió la cantidad, por ejemplo), respondés directamente sin /.

Buscar y consultar productos Vendedor

Ver todo el catálogo

Tú: /catalogo
Bot: 📋 Catálogo CGS Paraguay

    ELAION — Autos
      [20] ELAION F10 5W-30
      [21] ELAION F10 10W-40
        ...

Lista completa agrupada por categoría con sus IDs entre corchetes.

Ver la ficha de un producto

Si ya conocés el ID, escribilo directo con la barra:

Tú: /20
Bot: 🔧 [20] ELAION F10 5W-30
    Tecnología:     Semi-Sintético
    Viscosidad:     5W-30
    Specs:          API SN/CF
    Presentaciones: 1L · 4L · 20L
    Aplicaciones:   Motores nafta modernos
    🏷️ Más vendido

Buscar por nombre o tipo

Si no recordás el ID, podés buscar por nombre, viscosidad o tipo. Tolera errores de tipeo:

Tú: /buscar elaion 5w30
Bot: Encontré varios para "elaion 5w30":

    1. [20] ELAION F10 5W-30 — Semi-Sintético
    2. [23] ELAION F30 5W-30 — Sintético

    👉 Escribí 1 o 2 para la ficha completa.
Tolera typos: /buscar elaiom 5w30 (con "m" en vez de "n") también funciona. El bot detecta el error y sugiere el producto correcto.

Buscar por categoría

Atajos para ver una categoría completa:

ComandoCategoría
/auto · /autosAutos · ELAION
/moto · /motosMotos · RÖD
/camion · /camionesCamiones diesel · EXTRAVIDA
/otrosRefrigerantes · DOT 4 · Grasas · Hidráulico

Recomendar producto para un vehículo

Cuando el cliente te dice qué auto tiene y querés saber qué aceite usar:

Tú: /guia toyota corolla 2018
Bot: 🚗 Toyota Corolla (2014–2019) · nafta

    ✅ Recomendado: [20] ELAION F10 5W-30
    ↪ Alternativa:  [23] ELAION F30 5W-30

    📝 Motor 1.8L 2ZR-FE

    👉 /20 ficha · /vender 20 registrar
Truco: /buscar hilux también funciona. Si lo que escribís no es un producto, el bot lo busca como vehículo automáticamente.

Elegir de una lista

Cuando el bot te muestra una lista numerada, respondés con el número para elegir. No necesitás escribir la barra:

Tú: /buscar elaion
Bot: Encontré varios:
    1. [20] ELAION F10 5W-30
    2. [22] ELAION F30 5W-40
    3. [25] ELAION SUV 5W-40

Tú: 2
Bot: 🔧 [22] ELAION F30 5W-40 ...
4

Registrar ventas rápidas Vendedor

Para ventas de mostrador donde no necesitás identificar al cliente (consumidor final, atención rápida). Si la venta es para un cliente con RUC, usá el comando /pedido de la sección 5.

Forma fácil: el bot te guía

Si no querés acordarte de la sintaxis, escribí solo /vender y el bot te va preguntando paso a paso:

Tú: /vender
Bot: ¿Qué producto vendiste? Escribí el nombre o ID.

Tú: elaion 5w30
Bot: Encontré varios para "elaion 5w30":
    1. [20] ELAION F10 5W-30
    2. [23] ELAION F30 5W-30
    👉 ¿Cuál es? Escribí 1 o 2.

Tú: 1
Bot: ¿Cuántas unidades?

Tú: 3
Bot: ✅ Venta registrada
    [20] ELAION F10 5W-30 × 3 unidades
    🕐 14:32

Atajos rápidos (si ya sabés el ID)

ComandoQué hace
/vender 20Registra 1 unidad del producto [20]
/vender 20 3Registra 3 unidades del producto [20]
/vender 20 3, 26 1Multi-venta: 3 del [20] y 1 del [26], en un solo mensaje

Cancelar lo que estás haciendo

Si te equivocaste en el medio de un flujo, escribí /salir y el bot cancela todo. Después podés empezar de nuevo.

Tú: /salir
Bot: Listo, cancelé lo que tenías abierto. 👋
Cuidado: el bot no permite editar una venta ya registrada. Si registraste mal una venta, contactá al jefe para que la corrija desde el panel admin.
5

Pedidos con cliente Vendedor

Para ventas de ruta donde tenés que dejar registrado qué cliente compró cada producto. El bot identifica al cliente por su RUC y guarda el pedido vinculado a vos como vendedor.

Forma fácil: el bot te guía

Si no recordás el RUC del cliente o querés ver opciones, escribí solo /pedido:

Tú: /pedido
Bot: ¿Para qué cliente? Escribí RUC o nombre.

Tú: san lorenzo
Bot: Encontré varios clientes:
    1. AUTOREPUESTOS SAN LORENZO SRL · RUC 80012345-1
    2. LUBRICENTRO SAN LORENZO · RUC 80098765-3
    👉 Escribí 1 o 2 para elegir.

Tú: 1
Bot: ✅ Cliente: AUTOREPUESTOS SAN LORENZO SRL
    ¿Qué productos vas a cargar?
    Formato: ID cantidad, ID cantidad...
    Ej: 20 5, 26 2

Tú: 20 5, 26 2
Bot: Confirmá el pedido
    Cliente: AUTOREPUESTOS SAN LORENZO SRL
    RUC: 80012345-1 · San Lorenzo

      • [20] ELAION F10 5W-30 ×5
      • [26] EXTRAVIDA DX 15W-40 ×2

    Total: 7 unidades (2 productos)
    👉 Respondé si para confirmar o no para cancelar.

Tú: si
Bot: ✅ Pedido #1042 registrado
    Cliente: AUTOREPUESTOS SAN LORENZO SRL
    Total: 7 unidades · 2 productos
    🕐 14:32

Cliente nuevo: alta automática

Si el RUC que escribís no está cargado, el bot te lo dice y pregunta el nombre del cliente para darlo de alta en el momento:

Tú: /pedido 80099999-9
Bot: No tengo registrado el RUC 80099999-9.
    👉 Mandame el nombre / razón social del cliente para darlo de alta.

Tú: Lubricentro El Roble SRL
Bot: ✅ Cliente dado de alta: Lubricentro El Roble SRL
    ¿Qué productos vas a cargar?

Panel de administración Jefe

Toda la gestión de datos del sistema se hace desde el panel web en cgs-paraguay.netlify.app/admin.html. No hace falta saber SQL — son formularios normales.

Acceso

  1. Abrí cgs-paraguay.netlify.app/admin.html en cualquier navegador.
  2. Ingresá tu email y contraseña de admin.
  3. Vas a ver un sidebar con todas las secciones de gestión.
Sin sign-up público: nadie puede crearse cuenta. Las cuentas de admin se crean desde Supabase → Authentication → Users → Add user.

Productos

Catálogo de lubricantes que ve el bot y la landing pública. Campos clave: nombre, categoría (ELAION / EXTRAVIDA / RÖD / Otros), viscosidad, tecnología, specs, presentaciones, aplicaciones, badge, destacado.

Guía de Vehículos

Mapeo de qué lubricante recomendar para cada vehículo. Lo que devuelve el comando /guia del bot.

  • Marca y modelo: obligatorios (ej: Toyota Corolla)
  • Año desde / hasta: opcionales — vacío significa "sin restricción"
  • Tipo de motor: nafta / diesel / turbo / híbrido / moto 4T / moto 2T
  • Producto recomendado: obligatorio
  • Alternativa: opcional (para clientes que buscan algo más económico)
  • Notas técnicas: texto libre (ej: "Motor 1.8L 2ZR-FE")

Vendedores

Lista de quienes pueden usar el bot. Si alguien no figura acá, el bot no le responde.

  • Teléfono: formato internacional sin + ni espacios. Ej: 595981234567
  • Nombre: visible en los reportes
  • Categorías: qué tipo de productos vende (define qué notificaciones recibe en el futuro)
  • Ciudades: separadas por coma
  • Activo: tildá para que pueda usar el bot
Cómo dar de baja: destildar el "Activo" en vez de borrar la fila. Así se mantiene el historial de ventas y pedidos. Eliminar directamente falla si el vendedor tiene pedidos asociados.

El bot detecta los cambios en máximo 5 minutos (cache automático).

Clientes

Base de clientes identificados por RUC. Si bien el bot puede dar de alta clientes nuevos durante un pedido, desde acá podés:

  • Corregir typos en razones sociales que los vendedores cargaron rápido
  • Pre-cargar clientes habituales antes de la primera visita
  • Agregar contacto, teléfono, notas internas

El RUC es el identificador y no se puede modificar después de crear.

Pedidos

Visualización de todos los pedidos registrados por el equipo. Es solo lectura: los pedidos se crean exclusivamente desde el bot.

  • Filtrá por período: hoy, últimos 7 días, último mes, todo
  • Buscá por cliente, RUC o vendedor
  • Click en el ícono de ojo para ver el detalle: items, cantidades, notas

Configuración

Cambiar contraseña, configurar el botón de WhatsApp de la landing pública.

Problemas comunes

El bot no responde

  • ¿Pusiste el / al inicio? Sin barra, el bot no procesa el mensaje.
  • ¿Estás en la lista de autorizados? Si el bot te respondió alguna vez con "Este bot es de uso interno...", pedile acceso al admin.
  • ¿Estará el bot caído? Si el equipo entero deja de recibir respuestas, avisá al administrador.

Me dice "Este bot es de uso interno de CGS Paraguay"

Tu número no está cargado en la tabla de vendedores autorizados. Contactá al admin con tu número en formato internacional (sin +) y pedí que te dé de alta.

Quedé "trabado" en medio de un flujo

Escribí /salir. Cancela cualquier flujo activo y volvés al inicio.

No encuentro el producto que busco

  1. Probá /catalogo para ver todo lo que hay.
  2. Si lo que tenés es el nombre del vehículo del cliente, usá /guia en lugar de /buscar.
  3. Si igual no aparece, probablemente no está en el catálogo. Avisá al admin.

Mis ventas no aparecen en el reporte del día

El reporte muestra ventas desde las 00:00 (hora Paraguay). Si registraste una venta y la zona horaria está mal configurada, el admin lo puede ajustar.
